# Ping-Pang 需求参考

## 产品目标

让用户在打球时快速、准确、无思考成本地记分。打开第一眼即「精心设计过的专业运动工具」。

## 选手（当前）

- 选手 A / 选手 B，默认 **小乒** / **小乓**
- 赛前可自定义名称
- 未来：组队（多队、队员名、抽签/选对战）— 当前不实现

## 页面结构

```
┌─────────────────────┐
│   选手 A（可发球高亮） │  ← 点击 +1
├─────────────────────┤
│        球网          │
├─────────────────────┤
│   选手 B             │  ← 点击 +1
└─────────────────────┘
边缘：Undo | 重置 | 设置
```

## 小局规则

- 11 分制，领先 2 分胜
- 10:10 前：轮流发 2 球（T=0,1 → 首球方；T=2,3 → 对方；依此类推）
- 10:10 及后：轮流发 1 球，从常规规则下 T=20 时的发球方起交替
- 每小局记录 firstServer；局间建议交替首球方

## 大局规则

- 默认不开启
- 可选：三局两胜、五局三胜
- 小局 Win：明显但克制
- 大局 Win：更强反馈（🏆、更大动画、可 confetti）

## 持久化字段

- 小局 scoreA / scoreB
- gameWins、currentGameIndex
- matchFormat（none | bestOf3 | bestOf5）
- participant names
- firstServer、themeId

## PWA

- 添加到桌面、离线、全屏
- Wake Lock 防熄屏

## 未来扩展（勿在 MVP 实现）

历史记录、语音报分、分享战绩、云同步、七局四胜、自定义分数、组队模式
